# Euler v0.6

Welcome to **Euler v0.6**, a lightweight desktop assistant for Windows designed to execute everyday system tasks through natural language commands.

## Getting Started

1. Open the folder named 'dir'
2. Run **Euler.exe**.
3. Type your command into the input box and press **Enter** or click the **Send** button.

No additional installation is required.

## Current Features

* Open installed applications
* Close running applications
* Search the web
* Tell the current date
* Tell the current time
* Set timers
* Conversation-style interface

## Notes

* A **memory** folder will be created automatically after the first run. It stores log files generated by Euler.
* Do not move or delete files created inside the **memory** folder while Euler is running.
* Some commands depend on the applications installed on your computer. If an application cannot be found, Euler will notify you.

## Requirements

* Windows 10 or Windows 11

## Known Limitations

* Voice interaction is not available in this version.
* Responses are based on the current command system and are not AI-generated.
* Internet access is required for web searches.
